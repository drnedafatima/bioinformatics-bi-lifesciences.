Chapter 5: Machine Learning in Bioinformatics
Overview
This chapter covers supervised and unsupervised machine learning approaches applied to clinical and genomic data, forming the methodological bridge between raw biological data (Ch. 3–4) and advanced deep learning (Ch. 6).
Notebooks
Notebook	Topics Covered
`ch05_ml_bioinformatics.ipynb`	Classification (RF, SVM, LR), feature selection, PCA, clustering
Key Algorithms Demonstrated
Random Forest, Support Vector Machine, Logistic Regression
K-Means clustering for patient stratification
PCA and t-SNE for dimensionality reduction
ANOVA F-test for feature selection
Dataset Used
PIMA Indians Diabetes Dataset (CC0 licence)
TCGA-BRCA gene expression subset (see `datasets/` for download instructions)
